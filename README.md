# Analisis-Kinerja-Penjualan-TokoBuku

Proyek ini menganalisis data transaksi toko buku melalui tahapan data cleaning, feature engineering, dan analisis mendalam. Kami menemukan bahwa Total Pendapatan mencapai Rp 5.66 Miliar dengan tren pertumbuhan yang stabil sebelum adanya penurunan di akhir periode. Dari sisi produk, Buku Fiksi adalah pendorong utama pendapatan, sedangkan Alat Tulis mendominasi kuantitas penjualan. Analisis Box Plot Distribusi Harga juga mengonfirmasi adanya nilai transaksi outlier (harga premium) pada kategori buku.

Analisis RFM (Recency, Frequency, Monetary) mengidentifikasi segmen pelanggan kritis. Distribusi segmen  menunjukkan bahwa kelompok Champions adalah aset utama, sementara kelompok At Risk merupakan prioritas untuk diaktifkan kembali.

Rekomendasi Utama: Pertahankan loyalitas Champions dan segera lakukan kampanye win-back yang terpersonalisasi untuk segmen At Risk guna memaksimalkan lifetime value pelanggan.
